# Google Keep Clone

A responsive Google Keep inspired web app built with HTML, CSS and Vanilla JavaScript. The app allows users to create, add, archive and delete notes. Notes are automatically stored using the browser's local storage.

# Features 
- Create text notes
- Create checklist notes
- Archive and restore notes 
- Permanently delete notes
- Automatic saving using local storage 
- Responsive design for desktop and mobile devices 
- A few interactive tooltips
- Sidebar navigation

# Running the project
Use Visual Studio Code (VS code) with the "Live Server" extension.
1. Clone the repository
2. Open the project in VS code.
3. Install "Live Server" extension if you dont already have it.
4. Right-click anywhere on index.html file
5. Scroll down popup menu and select "Open with Live Server". The appication will open in your default browser.
